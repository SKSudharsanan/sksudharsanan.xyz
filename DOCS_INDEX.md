## üìã Quick Reference by Task

### "I just started, what do I do?"
1. Read [START_HERE.md](START_HERE.md)
2. Run `npm install` and `npm run dev`
3. Follow the priority checklist

### "How do I customize X?"
1. Check [CUSTOMIZATION.md](CUSTOMIZATION.md)
2. Find your section (Timeline, Projects, etc.)
3. Follow the examples

### "How do I deploy?"
1. Read [DEPLOYMENT.md](DEPLOYMENT.md)
2. Choose a platform (Vercel recommended)
3. Follow the steps (takes ~5 minutes)

### "Something broke, help!"
1. Check [SETUP.md](SETUP.md) troubleshooting section
2. Look at browser console
3. Try clean install: `rm -rf node_modules && npm install`

### "How does this work technically?"
1. Read [PROJECT_OVERVIEW.md](PROJECT_OVERVIEW.md)
2. Check component files in `src/`
3. Review data files in `src/data/`

### "I want to add a new feature"
1. Read [PROJECT_OVERVIEW.md](PROJECT_OVERVIEW.md) architecture
2. Check [CUSTOMIZATION.md](CUSTOMIZATION.md) for patterns
3. Create component in `src/components/`
4. Import in relevant section
